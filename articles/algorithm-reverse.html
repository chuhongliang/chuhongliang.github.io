整数反转
作者 chuhongliang | 发布于 2021-01-05
算法

给你一个 32 位的有符号整数 x ，返回 x 中每位上的数字反转后的结果。

如果反转后整数超过 32 位的有符号整数的范围[−231, 231 − 1]，就返回 0。

假设环境不允许存储 64 位整数（有符号或无符号）。

示例：

输入：x = 123
输出：321

function reverse(x) {
    let n = 0;
    while (x != 0) {
        n = n * 10 + x % 10;
        x = Math.floor(x / 10);
    }
    return Math.abs(n) > (2 ** 31) ? 0 : n;
}
